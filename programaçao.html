As TAGs mais utilizadas do HTML

-HTML- As tags HTML são elementos usados ​​para marcar e formatar os conteúdos: textual ou visual em documentos HTML.
Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada.

-HEAD- Essa TAG é usada como cabeçalho, ou seja, um conjunto de informações de uma página de um site.

-TITLE- Essa TAG define o título do documento, mostrado na barra de título de um navegador ou na aba da página.
conter somente texto e quaisquer marcações contidas no texto não são interpretadas.

-BODY- Essa TAG indica para o navegador o corpo do documento HTML.

-H1- Essa TAG é a mais importante da página e deve ser atribuída ao título principal do conteúdo.
As tags H2 são vinculadas aos subtítulos, as tags H3 a outros trechos de destaque (mas com menor relevância que as tags H2), e assim por diante.

-P- Essa TAG serve para a divisão dos parágrafos no seu código.
